# Imágenes reales — pendientes de autorización

Esta carpeta es el espacio reservado para fotografía real de Batmotrópicos
(equipo, sesiones, instalaciones, casos como Kokomo, etc.).

No se usan imágenes de stock en el sitio: los bloques que necesitan imagen
muestran un placeholder textual hasta que el material real esté autorizado.

Convención de nombres sugerida:
- `kokomo-01.jpg`, `kokomo-02.jpg` — caso Kokomo
- `equipo-01.jpg` — equipo / quiénes somos
- `sesion-01.jpg` — sesiones de formación / bienestar

Al añadir una imagen aquí, referénciala desde la página o componente
correspondiente con la ruta `/assets/img/real/<archivo>`.
